Provides a basic implementation of {@link net.jini.core.constraint.MethodConstraints} and a {@link net.jini.security.TrustVerifier} for common constraints. {@link net.jini.constraint.BasicMethodConstraints} is a basic implementation of MethodConstraints, allowing limited wildcard matching on method names and parameter types. {@link net.jini.constraint.ConstraintTrustVerifier} is a trust verifier for instances of the constraint classes defined in the {@link net.jini.core.constraint} package, and for the BasicMethodConstraints, {@link javax.security.auth.x500.X500Principal} and {@link javax.security.auth.kerberos.KerberosPrincipal} classes. @since 2.0 @version 2.0
