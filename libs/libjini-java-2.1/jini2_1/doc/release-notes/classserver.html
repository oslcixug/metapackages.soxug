Release Notes for ClassServer

ClassServer
v2.1 Release Notes

    Description

    ClassServer is a simple HTTP server for serving up JAR and class files.

    Changes since the v2.0.1 release

    Multiple Directory Support
    The class server can now serve up files from multiple directories. The -dir (or -dirs) command line option and the dirlist constructor parameter now accept a list of directories to serve files from, with entries separated by the system path-separator character.

    Executable JAR file
    There is a new executable JAR file, classserver.jar, that can be used to run the class server.
