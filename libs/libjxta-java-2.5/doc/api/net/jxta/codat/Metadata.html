Class Metadata

java.lang.Object
  net.jxta.codat.Codat
      net.jxta.codat.Metadata

public class Metadata
extends Codat

Medata Codats are special codats that contain information about another Codat. Multiple medata Codats can refer to the same Codat. Medata codats can hold any kind of information about a codat, such as a symbolic name, description, index and searching information, etc.
