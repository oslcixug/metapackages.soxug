Metadata
public class Codat
extends Object

The common container for managing content within JXTA. A Codat consists of:

  • Content data for the Codat in the form of a JXTA Document.
  • A persistent canonical identifier for the Codat in the form of a CodatID.
  • An optional CodatID for an associated Codat for which this Codat is metadata.
