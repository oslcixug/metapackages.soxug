Quen desenvolve este proxecto?

A promoción do software libre no ámbito universitario coordínase a través da Oficina de Software Libre que depende actualmente do CONSORCIO para o desenvolvemento de sistemas para Xestión Universitaria, CIXUG, que desde mediados de 2012 asumiu esta tarefa cun único equipo distribuído entre as tres universidades que comparten así unha mesma programación e recursos.
