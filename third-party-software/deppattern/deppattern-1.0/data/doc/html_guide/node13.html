Corrector (-c)

Option -c allows dp.sh to generate a file with the same format as the input (i.e., a tagged text). The aim is to use specific syntactic rules to make morpho-syntactic corrections. This option is useful to identify and correct systematic errors of PoS taggers using grammatical rules. More information can be found in Chapter 2.2 of the Tutorial.
