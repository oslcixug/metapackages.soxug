Options for Different Output Formats

The output is in raw plain text. In further versions, we'll provide more elaborate output extensions (xml, html, ...). By now, we provide 3 different ways of visualizing the resulting data: basic representation of dependency analysis (-a), full representation of dependency analysis (-fa), and PoS tag representation (-c).
