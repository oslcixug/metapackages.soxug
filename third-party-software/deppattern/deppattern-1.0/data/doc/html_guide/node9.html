Grammar File

The file containing the grammar must be in plain text format. Below, you'll find a toy example of a grammar with 4 dependency-based rules:

AdjnR: NOUN ADJ

Agr: number, genre

SpecL: DT NOUN

Agr: number, genre

SubjL: NOUN [ADV]* VERB

Agr: number

DobjR: VERB [ADV]* NOUN

To learn more about DepPattern formalism, look up the tutorial in the doc directory.
